# 📡 Telco Customer Churn Predictor
An end-to-end Machine Learning web application deployed to predict customer attrition using historical billing and demographic data.

### 🧠 System Architecture
* **Algorithm:** Random Forest Classifier (Ensemble Learning)
* **Data Processing:** Pandas (One-Hot Encoding, Missing Value Imputation)
* **Model Serialization:** Joblib (Pickle)
* **Frontend/Deployment:** Streamlit Community Cloud

### ⚙️ How it Works
The AI model was trained on the official Telco Customer dataset. It analyzes four critical features (Tenure, Contract Type, Internet Service, and Monthly Charges) to calculate a real-time risk factor indicating the likelihood of a customer canceling their subscription.
